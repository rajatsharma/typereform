# Typereform

_My NextJs + Typescript Starter with focus on static site generation (Work in Progress)_

## Usage

- Start dev server:

```sh
yarn dev
```

- Build for production:

```sh
yarn build
```

- Generate static site:

```sh
yarn export
```
